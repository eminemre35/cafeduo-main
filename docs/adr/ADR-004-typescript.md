# ADR-004: React + TypeScript Frontend

## Status
Accepted

## Context: Type safety gereksinimi

## Decision
TypeScript 5.0 ile React 18.

### Nedenleri
- Compile-time hata yakalama
- IDE intellisense
- Refactoring güvenliği
- Team collaboration

## Consequences
- Learning curve (junior devs)
- Build time artışı (minimal)
- Daha az runtime hatası

## Alternatives
- JavaScript + JSDoc - Rejected
- Flow - Rejected (community support)
