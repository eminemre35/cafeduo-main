## ADR Formatı

Her ADR şu bölümleri içerir:
- **Status**: Accepted, Deprecated, Superseded
- **Context**: Kararın alındığı durum
- **Decision**: Ne karar verildi
- **Consequences**: Pozitif ve negatif etkiler
- **Alternatives**: Değerlendirilen alternatifler
